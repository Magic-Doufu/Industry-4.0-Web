# Arduino FC-37雨滴模組感測器

首先將FC-37接上藍色小板子，由於FC-37的接點上並沒有標示正負極，所以就先隨便接上。 接著藍色小板子四個接點的部份分別是VOC, GND, D0, A0

將VOC接到Arduino板子的5V，GND接到Arduino板子的GND。 最後將A0接到Arduino板子的A0 ~ A5其中一個接點即可。

程式範例:

void setup() {
  // put your setup code here, to run once:
Serial.begin(9600);
}

void loop() {
  // put your main code here, to run repeatedly:
int rain = analogRead(A1);
if (rain<1000)
Serial.println("wet");
else
Serial.println("dry");
delay(1000);

}
