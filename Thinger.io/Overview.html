硬體設備

本節內容將介紹如何透過不同平台的不同開發板來加入Thinger.io平台，使其成為雲端上的一員。

目前支援的環境可以相容ESP32/8266、Arduino、Sigfox、運行Linux的設備如Raspberry Pi、Intel Edison等。

Arduino


幾乎可用於相容Arduino IDE的任何硬體，只要你在IDE中可以新增的硬體幾乎都適用。
除了原生Arduino以外，ESP8266、CC3200、Arduino-WiFi、BPI UNO(ESP32)皆適用，僅須先行建構開發板於Arduino IDE中的環境。
當然，您仍然可以使用任何其他具有Wifi或乙太網路功能的Arduino板。

Linux


如果您想在基於運行Linux OS的設備開發物聯網產品，此平台還有一個特殊客戶端。
每個常見的Linux板都有一些特定的範例，如Raspberry Pi或Intel Edison。
幾乎任何運行Linux發行版的計算機都可以運行客戶端，包括帶有Mac OS的電腦。

如何在APP中管理Thinger.io？

伺服器 API


Thinger.io Cloud由REST API提供支援，旨在將您的IoT項目輕鬆集成到其他應用程序，如Web服務、手機或桌面應用程序。
因此，如果您想將您的裝置與您自己的應用程序或服務集成，那麼這就是起點。
