Arduino

原文參照：https://docs.thinger.io/devices/arduino

本文檔與Thinger.io平台的Arduino客戶端程式庫版本相關。使用此程式庫，您可以使用乙太網，Wifi，GSM連接任何Arduino開發板或其他此受支援的開發板（如ESP8266，NodeMCU和TI CC3200）。客戶端程式庫允許將您的IoT設備連接到Thinger.io雲端平台。這是專為Arduino IDE設計的程式庫，因此您可以輕鬆編程設備並在幾分鐘內連接它們。此平台支援多種網絡接口和開發板，如Ethernet Shield，Wifi Shield和GSM。

它還支援其他開發板，如ESP8266（或NodeMCU），TI CC3200 Launchpad 和 Adafruit CC3000，此平台的Arduino IDE最低需求為版本1.6.3。

需求

  • Arduino IDE
  • 相容於 ArduinoIDE 的開發板
  • Thinger.io 程式庫

Arduino IDE

此平台的Arduino IDE最低需求為版本1.6.3，且需要支援「程式庫管理器」，如您尚未安裝Arduino IDE可由官方網站下載最新版。

Arduino IDE 下載頁

接下來將為您介紹取得程式庫的方法，建議使用程式庫管理器，此方法極大簡化了搜尋與安裝過程，且會隨著官方發布新版時提供更新。

安裝程式庫

開始構建 thinger.io 設備的第一步是在 Arduino IDE 中安裝所需的程式庫，以支援披露設備擁有的資源，如傳感器值，燈光，繼電器等。

程式庫管理器

在Arduino IDE中安裝程式庫的最簡單方法是使用程式庫管理器。要安裝thinger.io程式庫，請按照以下步驟操作：

打開程式庫管理器：在Arduino的選單中打開程式庫管理器 工具 > 管理程式庫


搜尋並安裝 thinger.io 程式庫


使用名稱 thinger.io 搜尋，然後單擊 Install 進行安裝。官方發布更新後，您還可以從此管理器更新程式庫。
