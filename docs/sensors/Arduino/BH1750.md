# GY-302 BH1750 光強度光照度感測器模組 (數位輸出)

感測晶片：ROHM原裝BH1750FVI晶片
電源：DC3 ~ 5V
感測範圍：1 ~ 65535 Lux
感測器內置16bit，AD轉換器直接數位輸出，省略複雜的計算
不區分環境光源、接近於視覺靈敏度的分光特性
可對廣泛的亮度進行1Lux的高精度測定
具有低電流斷電保護
受到紅外線影響非常小

型號:GY-302
尺寸：13.9mm X 18.5mm
採用ROHM原裝BH1750FVI晶片
供電電源? ：3-5v
資料範圍：0-65535
感測器內置16bitAD轉換器
直接數位輸出，省略複雜的計算，省略標定
不區分環境光源
接近於視覺靈敏度的分光特性
可對廣泛的亮度進行1勒克斯的高精度測

程式範例:(需另行下載BH1750 library)
```cpp
#include <Wire.h>
#include <BH1750.h>

BH1750 lightMeter;

void setup(){

  Serial.begin(9600);

  // Initialize the I2C bus (BH1750 library doesn&#39;t do this automatically)
  // On esp8266 devices you can select SCL and SDA pins using Wire.begin(D4, D3);
  Wire.begin();

  lightMeter.begin();
  Serial.println(F("BH1750 Test"));

}

void loop() {

  float lux = lightMeter.readLightLevel();
  Serial.print("Light: ");
  Serial.print(lux);
  Serial.println(" lx");
  delay(1000);

}
```
